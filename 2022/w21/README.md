# Week 21 challenge

Write a function `iStandOut` which accepts an array of numbers. All numbers will be either even, or odd. Except one. 


Examples:
```
iStandOut([1,19,23,5,4,9,13]) // returns 4
iStandOut([8,16,2,11,88,24]) // returns 11
```
